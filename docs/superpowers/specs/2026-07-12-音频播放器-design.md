# 音频播放器 设计文档

## 概述

基于 macOS SwiftUI 的音频播放器，支持 WAV/MP3/AAC 格式，文件夹浏览、声纹显示、高振幅段落自动框选与批量导出。

---

## 整体架构: 单窗口 MVVM + AVFoundation

```
┌─────────────────────────────────────────────────────────┐
│  工具栏 (播放/暂停/停止/音量/导入/导出/灵敏度滑块/设置)   │
├──────────────┬──────────────────────────────────────────┤
│  左侧边栏     │         主播放区                          │
│  ┌─────────┐ │  ┌─────────────────────────────────────┐ │
│  │浏览文件夹 │ │  │        声纹显示区 (Canvas)           │ │
│  │过滤输入框 │ │  │  静态波形 + 播放头 + 段落框选        │ │
│  ├─────────┤ │  │                                     │ │
│  │文件列表  │ │  ├─────────────────────────────────────┤ │
│  │(分页20)  │ │  │ 进度条 | 文件名 | 当前位置 | 总时长   │ │
│  └─────────┘ │  └─────────────────────────────────────┘ │
├──────────────┴──────────────────────────────────────────┤
│  底侧栏 — 状态/日志 (解码... / 导出完成 / 错误信息)       │
└─────────────────────────────────────────────────────────┘
```

### 数据流

```
AVAudioFile (解码)
    ↓ PCM buffer
AudioAnalyzer — 降采样 → 波形点数组 + 段落检测
    ↓
AppViewModel (唯一数据源)
    ├→ SidebarView      文件列表/过滤
    ├→ WaveformView     Canvas 渲染
    ├→ PlaybackBar      播放控件
    ├→ ToolbarView      工具栏按钮
    └→ StatusBarView    日志输出
```

### 1. AppViewModel (唯一状态管理器)

`@Observable` class，作为整个应用的单一数据源。

```
Fields:
  - openFolderURL: URL?          当前打开的文件夹
  - allFileItems: [AudioFileItem]  扫描到的所有文件

  - currentPage: Int             当前分页 (0-based)
  - selectedFile: AudioFileItem? 选中的文件
  - playingFile: AudioFileItem?  正在播放的文件
  
  - isPlaying: Bool
  - isPaused: Bool
  - currentTime: TimeInterval
  - duration: TimeInterval
  - volume: Float                0.0 ~ 1.0
  
  - waveformSamples: [Float]     归一化振幅数组 (-1.0 ~ 1.0)
  - segments: [AudioSegment]     检测到的段落

  - logs: [LogEntry]             日志列表
```

## 段落检测算法

### 自适应阈值公式

```
threshold = mean(amplitudes) × sensitivityFactor
sensitivityFactor ∈ [1.0, 5.0], 默认 2.0
```

### 段落合并规则

1. 遍历降采样后的振幅数组，找出所有连续超过阈值的区间
2. 相邻区间间隔 < 0.5 秒的合并为一个段落
3. 时长 < 0.3 秒的忽略（可能是噪音尖峰）
4. 将采样索引转换回实际时间：`time = index × (duration / totalSamples)`

## 波形渲染

### 降采样策略

- `AVAudioPCMBuffer` 获取 full-scale float buffer
- 将样本点分成 N 个桶（N = Canvas 宽度），每桶取 `max(abs(sample))`
- 归一化为 0.0 ~ 1.0（占据 Canvas height 的 80%，上下留白 10%）

### Canvas 渲染顺序

```
1. 背景色
2. 波形路径 (Path, 顶部 + 底部镜像)
3. 段落高亮矩形 (GeometryReader 计算 x 坐标)
4. 播放头竖线 (位置随进度更新)
5. 选中段落边框加粗
```

### 点击交互

- 点击段落矩形 → toggle 选中 (Cmd 累加)
- 点击波形空白处 → 跳转到对应时间

## 音频裁剪导出

### 导出流程

1. 构建 `AVAssetExportSession`
2. `timeRange = CMTimeRange(start: startCMTime, duration: endCMTime - startCMTime)`
3. 输出格式与原文件一致
4. 输出路径：`out/$原名称_00:35-02:08.后缀`（起始-结束 mm:ss 格式）
5. `out/` 目录位于 workspace（项目根目录），不存在则自动创建
6. 批量导出串行执行，底侧栏输出进度日志
7. 导出完成后弹出 Finder 窗口定位到 `out/` 目录

### 错误处理

| 场景 | 行为 |
|------|------|
| 文件不可读/损坏 | 跳过，日志输出警告 |
| 导出磁盘不足 | 中断导出，日志输出错误 |
| 不支持格式误入 | 跳过，日志输出警告 |
| 文件夹为空 | 列表显示空状态 |

## 文件扫描与分页

- 选中文件夹后，深度优先扫描（不递归子目录）
- 按文件名排序
- 前 20 个文件立即渲染，后续按需加载
- 加载中显示 "加载中..."

---

## 播放控制

- `AVAudioPlayer` 负责播放
- 音量独立于系统音量（`player.volume`）
- 播放完毕自动停止，playingFile 保持不变，播放头回到起点
- 双击文件列表项 → 开始播放
- 单击文件列表项 → 加载波形 + 选中

## 技术决策总结

| 决策点 | 选择 |
|--------|------|
| 音频格式 | WAV, MP3, AAC |
| 文件浏览 | 选择文件夹，一次性扫描 |
| 声纹效果 | 静态波形 + 播放头 + 段落框选 |
| 段落检测 | 自适应阈值 + 手动灵敏度调节 |
| 段落选择 | 单击/ Cmd+Click 多选 + 全选导出 |
| 段落命名 | `out/$原名称_mm:ss-mm:ss.后缀` |
| 过滤 | 模糊匹配文件名 |
| 底侧栏 | 操作日志 |
| 播放结束 | 自动停止 |
| 菜单栏 | 应用内自定义工具栏 |
